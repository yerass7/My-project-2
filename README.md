# User Management System

## Course
Application Design Patterns  
Прак2 – Design Principles

## Description
Простой пример системы управления пользователями для небольшой компании.
Проект демонстрирует применение принципов YAGNI, KISS и DRY.

## Features
- Добавление пользователей
- Обновление данных пользователя
- Удаление пользователей

## Principles Used
- YAGNI – реализован только необходимый функционал
- KISS – простой и понятный код без лишних абстракций
- DRY – повторяющаяся логика вынесена в отдельные методы
